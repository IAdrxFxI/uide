# Generador Seguro de Contraseñas En Python

Objetivo del programa:
Desarrollar un generador de contraseñas seguras que permita al usuario crear contraseñas personalizadas según la longitud y los tipos de caracteres que desee incluir.

El objetivo principal es aplicar estructuras básicas de programación en Python como:

1. Entrada de datos

2. Condicionales

3. Estructuras repetitivas

4. Uso de librerías (random)

Funcionalidades Principales:
El programa permite:

1. Seleccionar la longitud de la contraseña:

El usuario ingresa la cantidad de caracteres que desea que tenga su contraseña.

2️: Elegir tipos de caracteres:

El usuario puede decidir si desea incluir:

Letras mayúsculas

Números

Símbolos especiales

Las letras minúsculas siempre están incluidas por defecto.

3. Generación aleatoria:

Se utiliza la librería random de Python para seleccionar caracteres aleatorios y construir la contraseña mediante una estructura repetitiva (for).

4. Mostrar resultado:

El sistema imprime en pantalla la contraseña generada.
